# Hospital Management System

A comprehensive Hospital Management System database project designed to organize and manage hospital operations efficiently.

## Project Structure

- `docs/` : Documentation, ERD diagrams, flowcharts
- `src/` : Source code (frontend + backend)
- `tests/` : Unit and integration tests
- `sql/` : Database scripts (schema, seeds, migrations)
- `.gitignore` : Git ignore rules
- `LICENSE` : MIT License
- `CONTRIBUTING.md` : Contribution guidelines
